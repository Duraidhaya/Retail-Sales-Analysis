# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data using Excel, SQL, Python, and Power BI.

## Tools Used
- Microsoft Excel
- MySQL
- Python
- Pandas
- Matplotlib
- Power BI

## Dataset
Sample Superstore Dataset

## Key Insights
- Technology category generated the highest sales.
- West region generated the highest profit.
- Consumer segment generated the highest sales.
- Standard Class shipping produced the highest profit.

## Project Workflow
Dataset → Excel → SQL → Python → Power BI Dashboard
