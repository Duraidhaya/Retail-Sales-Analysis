# Retail Sales Analysis

## Project Overview
This project analyzes retail sales data using Excel, SQL, Python, and Power BI.

## Tools Used
- Excel
- SQL
- Python (Pandas, Matplotlib)
- Power BI

## Project Structure
01_Dataset/
02_Excel/
03_SQL/
04_Python/
05_PowerBI/
06_Screenshots/

## Key Insights
- Technology generated the highest sales.
- West region earned the highest profit.
- Consumer segment contributed the highest sales.
- Standard Class generated the highest profit.
